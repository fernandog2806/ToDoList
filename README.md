# 📝 To-Do List Dinámica con EJS

Proyecto de lista de tareas interactiva enfocado en la manipulación dinámica del DOM y persistencia de datos.

### 🚀 Funcionalidades
- **Persistencia**: Uso de `localStorage` para mantener las tareas tras recargar.
- **Bulk Actions**: Checkbox maestro para seleccionar y eliminar tareas en bloque.
- **UI Reactiva**: Los controles de acción solo aparecen cuando hay contenido.

### 🛠️ Tecnologías
- **Backend**: Node.js / Express
- **Frontend**: EJS / CSS3 / JavaScript Vanilla

### 📂 Estructura
- `/public`: Activos estáticos (CSS y JS del cliente).
- `/views`: Plantillas dinámicas de EJS.
- `app.js`: Servidor central.
